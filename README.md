# 💖 Text-Based Romance Game

A simple text-based romance simulator built using Python.

## 🎮 Features
- Player creation system
- Interactive choices
- Basic story progression
- Multiple character interactions

## 🛠 Tech Used
- Python

## 🚀 How to Run
1. Download the project files  
2. Run `main.py`  
3. Follow the on-screen choices
